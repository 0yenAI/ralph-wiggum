【タイトル】
【Gemini CLI連携】自律反復型AIエージェント・コントローラー "Ralph Wiggum for Gemini"

【キャッチコピー】
「一回でダメなら、できるまでやらせればいい。」
あなたのAIアシスタントを、粘り強い専属エンジニアに変える魔法のスクリプト。

【製品概要】
AIにコード修正を頼んでも、一発で完璧に直らないことはありませんか？
"Ralph Wiggum" は、Gemini CLI や Google IDE Antigravity などのAIツールと連携し、
指定された「完了シグナル」が出るまで、何度でもタスクを再試行させるためのコントローラーです。

エラーが出たら、その出力を元にAIが再考し、修正する。
この「自律的な反復ループ」を、たった1つの軽量なPythonスクリプトで実現します。

【こんな人におすすめ】
- Gemini CLI を使って開発をしているが、手動での指示出しに疲れた方
- テスト駆動開発(TDD)をしていて、テストが通るまでAIに修正を続けさせたい方
- 寝ている間にリファクタリングを終わらせておきたい方
- Google IDE Antigravity 環境をフル活用したい方

【機能・特徴】
- **シンプル＆パワフル**: 複雑な設定ファイルは不要。コマンドライン引数だけで動作します。
- **クロスプラットフォーム**: Windows, Mac, Linux すべての環境で動作確認済み。
- **柔軟な連携**: Gemini CLI だけでなく、コマンドラインで動作するあらゆるAIツール（Claude CLI, ChatGPT CLIなど）のラッパーとして機能します。
- **完了保証ロジック**: 「ALL GREEN」「FIXED」などの合言葉を検知するまで、AIを解放しません。

【使い方（例）】
以下のコマンドを打つだけで、バグ修正が終わるまでAIが働き続けます。
`python ralph_gemini.py "バグを直して。完了したら DONE と言って" --completion-promise "DONE"`

【動作環境】
- Python 3.8 以上
- 任意のAI CLIツール (Gemini CLI 推奨)
